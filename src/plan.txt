4️⃣ UNION / FLEET SIDE — QUEUE CONTROL (WEB)

This is where power + transparency coexist.

UNION QUEUE MANAGEMENT PAGE
A. Queue Definition

Union can define queues based on:

Route
Truck type
Cargo category
Priority flags
Example:
Queue: Mumbai → Pune | 16T | FMCG

B. Queue View (Ordered List)

Shows:

Driver name
Truck ID
Last trip completed
Queue position
Status (Available / On Trip / Skipped)

Union can:

Freeze queue
Reorder (with audit log)
Remove driver (disciplinary)

C. Job Injection

When shipment arrives:
Union assigns it to a queue, not a driver
System starts offering it top-down

5️⃣ RULE ENGINE (THIS IS IMPORTANT)

Queues are not hardcoded logic.
Union controls rules like:
Max skips allowed
Skip penalty (end of queue / temporary block)
Timeout duration (e.g. 10 minutes)
Emergency override
📌 These rules are data-driven, not code-driven.
